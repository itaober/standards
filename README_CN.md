## 介绍

个人前端开发规范及工具库，包括:

- [ESLint](./packages/eslint/README_CN.md)：TODO: 描述，npm 包为 `@itaober/eslint-config`；
- [Prettier](./packages/prettier/README_CN.md)：提供了一组预定义的 Prettier 配置，帮助快速统一代码格式，提升代码可读性和可维护性，npm 包为 `@itaober/prettier-config`；
- [TSConfig](./packages/tsconfig/README_CN.md)：提供了一套适用于不同类型项目的 TypeScript 配置文件，npm 包为 `@itaober/tsconfig`；
- [Commitlint](./packages/commitlint/README_CN.md)：提供了一组预定义的 Commitlint 配置，帮助统一代码提交规范，并支持命令行 prompt 提示，npm 包为 `@itaober/commitlint`：

## 安装

```bash
pnpm add @itaober/eslint-config @itaober/prettier-config @itaober/tsconfig @itaober/commitlint -D
```
